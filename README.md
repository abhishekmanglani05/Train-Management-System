🚆 Train Management System (Servlet-Based Web Application)

A full-stack Java Servlet-based web application that implements a complete Train Management System with essential CRUD functionalities. This project demonstrates backend development using Servlets, JDBC, and database integration for handling real-world operations like user management and ticket booking.

🔑 Key Features
-👤 User Authentication
-User Registration
-Secure Login & Logout
-🎫 Ticket Management
-Book Train Tickets
-View Booked Tickets
-Delete Tickets


🛠️ User Profile Management
Update User Details
🔄 CRUD Operations
Fully implemented Create, Read, Update, Delete functionalities
⚙️ Tech Stack
Java (Servlets & JSP)
JDBC
Apache Tomcat Server
Derby / SQL Database
HTML, CSS



💡 Highlights
Implements MVC-like structure using Servlets
Handles session management for authentication
Demonstrates database connectivity and transaction handling
Beginner-friendly project showcasing core web development concepts in Java
